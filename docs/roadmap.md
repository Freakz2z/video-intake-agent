# Roadmap

The project focuses on practical post-recording organization inside Codex and Claude Code.

## Shipped in v0.2

- Timestamped storyboard frames and Agent-written content timelines.
- Local visual fingerprints and similar-video clustering.
- Reviewable, integrity-protected archive plans.

## v0.3 — Recording review notes

- Create an opt-in Markdown or JSONL issue log per recording.
- Record relative timestamp, category, severity, description, and optional screenshot reference.
- Keep edits proposal-based and never write a log without confirmation.

## v0.4 — Better naming evidence

- Add optional local speech transcription.
- Support imported transcripts from external services without storing API keys.
- Include transcript excerpts in the Agent evidence bundle.

## v0.5 — High-volume workflows

- Folder watching with an explicit review queue.
- Configurable filename templates and project-level defaults.
- Persistent fingerprint caching and exact duplicate detection.

## Non-goals

- Replacing an NLE or video editor.
- Uploading video data to a cloud service by default.
- Renaming recordings without an auditable plan and explicit confirmation.
