# Changelog

All notable changes are documented here.

## Unreleased

- Added a persistent local fingerprint cache with automatic stat/sample invalidation and cache telemetry.
- Added selective SHA-256 detection of byte-identical videos alongside heuristic visual groups.

## 0.2.0 — 2026-07-12

- Added timestamped storyboard extraction and Markdown timeline rendering.
- Added local visual fingerprints and configurable complete-link similarity grouping.
- Added integrity-protected archive plans with collision checks, confirmation, and rollback attempts.
- Expanded Codex and Claude Code workflows for timeline descriptions and similarity archiving.
- Added a reproducible, openly licensed Big Buck Bunny demo with three small test clips.
- Added `video-intake <video-or-directory>` as the one-command preparation entry point.
- Added package metadata and repeatable source-build validation.
- Added `doctor`, version reporting, privacy documentation, and an architecture overview.
- Reworked the public README around the one-command local workflow and open demo only.
- Added a reusable SVG logo, centered project header, bilingual navigation, a complete Simplified Chinese README, and Chinese contribution guidance.

## 0.1.0 — 2026-07-12

- Added FFprobe-based recording inspection and FFmpeg contact sheets.
- Added safe single-video rename proposals and confirmed execution.
- Added directory scanning, batch rename plans, collision checks, and rollback attempts.
- Added Codex skill and Claude Code command templates.
- Added tests, linting, GitHub Actions CI, security guidance, and contribution files.
